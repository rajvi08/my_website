Digital Tools

Organize It!

As apartments and homes get smaller and more expensive, people need affordable, customizable spaces. This solution provides a DIY design tool that simplifies closet customization, maximizes space, and fits users' budgets. Organize It! is a tool in the form of a set of Grasshopper plug-ins that lets users design custom closets.

InBetween

This project draws inspiration from “Browsing the Latent Space: A New Approach to Interactive Design Exploration for Volumetric Generative Systems.” Aim: To create a tool that will help the user visualize the latent space to generate a wide range of design iterations as part of the initial conceptual phase.
